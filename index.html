Outfitter for All Seasons!

This is a website that showcases how all four seasons are unique in their own ways. On the top left corner you'll see a header(with my name on it) written in bold cursive. Sitting above this paragraph are four links, and if you click on these links it'll transfer you to a new page. Each page includes an image, logo, and background color. These unique attributes are suppose to represent the characteristics of each particular season.

While creating this website I used JavaScript and CSS to style and design my work. In my html file I refrenced jQuery from an outside source and then linked it to my JavaScript file. jQuery is a feature in the JavaScript library and is used as a shortcut to make changes on the html file.
